Projects

Multi-Media Video Summarization Using Parallel Deep Semantic Features The allocation of significant score to specific clips or frames in a raw video is a critical task to make a summary. Previous work have lot on this topic, using minimal visual features. In this project, we propose a novel and generic framework for video summarization by collecting different static and motion features, and input them to attention mechanism for summarized video. First step is dataset generation, which input to the second stage. In second step, Attention mechanism is implemented which followed by assignment of weights to significant features and multi-source fusion, and in final step H5 file (summarized features) convert into a video using OpenCv and knapsack algorithm. Experimental evaluation have been done using two benchmark datasets, SumMe and TvSum. By evaluating on these datasets, we improve the results using parallel deep semantic features.

Chat with Web using DeepSeek Our project, DeepSeek Web Assistant, is a powerful AI-driven browser extension designed to make web browsing more interactive and insightful. With this extension, users can: Summarize web pages instantly using DeepSeek's AI models, Ask questions about web content and receive relevant answers, Use the "Highlight to Ask" feature, allowing them to highlight text and ask questions related to the selected content, Interact with research papers, articles, and general web pages to extract key insights efficiently, Seamlessly integrate AI-powered search and summarization into their browsing experience. This tool is particularly useful for students, researchers, and professionals who need quick, precise information while browsing the web.
